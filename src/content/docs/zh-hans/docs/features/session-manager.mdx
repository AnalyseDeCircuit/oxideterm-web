**Session Manager** 提供类似电子表格的完整视图，展示所有已保存的 SSH 连接。它是侧边栏树视图的有力补充，支持强大的搜索、筛选、批量操作和标签管理。

## 打开 Session Manager

点击侧边栏的 "Session Manager"（表格图标），或使用命令面板（`⌘K`）。

## 连接表格

表格以可排序的列展示所有已保存连接：

| 列 | 说明 |
|----|------|
| **名称** | 连接显示名称 |
| **主机** | 主机名或 IP 地址 |
| **端口** | SSH 端口（默认 22） |
| **用户名** | 登录用户名 |
| **认证** | 认证类型（密码 / 密钥 / Agent） |
| **分组** | 文件夹分组 |
| **标签** | 自由格式标签 |
| **上次使用** | 最后连接的相对时间戳 |
| **操作** | 连接、编辑、复制、删除 |

点击列标题排序，按住 `Shift` 可多列排序。

## 搜索与筛选

工具栏搜索框在名称、主机、用户名和标签中**实时模糊搜索**，输入即刻更新结果。

使用左侧**文件夹树**按分组筛选：
- **所有连接** — 显示全部
- **未分组** — 未分配分组的连接
- **最近使用** — 7 天内使用过的连接
- 命名分组及子分组

## 标签

**标签**是附加在连接上的自由格式标签，用于临时分组和筛选。

- 通过"管理标签"行操作添加或移除标签
- 表格中以彩色标签chip展示
- 搜索包含标签内容

常用标签示例：`prod`、`staging`、`dev`、`datacenter-eu`、`team-backend`。

## 批量操作

使用复选框多选（表头复选框 = 全选可见项）：

| 批量操作 | 说明 |
|---------|------|
| **删除所选** | 一次删除多个连接 |
| **移至分组** | 将多个连接重新分配到某分组 |
| **添加标签** | 为所有所选连接应用标签 |

选中行后工具栏显示计数。批量操作执行前有确认对话框。

## 分组管理

左侧文件夹树管理连接分组：

- **新建子组** — 创建嵌套分组层级
- **重命名** — 原地重命名
- **删除分组** — 先将其中的连接移至未分组再删除
- 从表格拖拽连接到分组以重新分配

## 编辑连接

点击任意行的**编辑**打开连接编辑器：
- 所有连接属性可内联编辑
- 认证方式可在密码 / 密钥 / Agent 间切换
- 密钥文件路径可通过文件选择器浏览
- 分组分配可从选择器中选取

## 导入/导出

使用工具栏**导入** / **导出**按钮操作 `.oxide` 加密文件，详见[连接管理](/zh-hans/docs/features/connections)中的加密规格。

**自动路由**按钮分析现有会话，自动为新连接推荐最优 ProxyJump 路径。
