OxideTerm 所有可用键盘快捷键的完整参考。

## 应用与窗口

| 操作 | macOS | Windows / Linux |
|------|-------|-----------------|
| 新建本地终端 | `⌘ T` | `Ctrl+T` |
| Shell 选择器 | `⌘ ⇧T` | `Ctrl+Shift+T` |
| 新建 SSH 连接 | `⌘ N` | `Ctrl+N` |
| 关闭当前标签 | `⌘ W` | `Ctrl+W` |
| 下一标签 | `⌘ }` | `Ctrl+Tab` |
| 上一标签 | `⌘ {` | `Ctrl+Shift+Tab` |
| 跳到第 1–9 标签 | `⌘ 1`–`⌘ 9` | `Ctrl+1`–`Ctrl+9` |
| 命令面板 | `⌘ K` | `Ctrl+K` |
| 切换侧边栏 | `⌘ \` | `Ctrl+\` |
| 打开设置 | `⌘ ,` | `Ctrl+,` |
| Zen 模式 | `⌘ ⇧Z` | `Ctrl+Shift+Z` |
| 快捷键帮助 | `⌘ /` | `Ctrl+/` |

### 命令面板补充

以下操作可通过命令面板访问：

| 操作 | macOS | Windows / Linux |
|------|-------|-----------------|
| 切换底部面板 | `⌘ J` | `Ctrl+J` |
| 广播输入 | `⌘ B` | `Ctrl+B` |
| AI 侧边栏 | `⌘ ⇧A` | `Ctrl+Shift+A` |

## 终端内

以下快捷键在终端窗格聚焦时生效：

| 操作 | macOS | Windows / Linux |
|------|-------|-----------------|
| 搜索 | `⌘ F` | `Ctrl+Shift+F` (Win) / `Ctrl+F` (Lin) |
| 内联 AI 面板 | `⌘ I` | `Ctrl+Shift+I` |
| 关闭面板 | `Esc` | `Esc` |
| 复制 | `⌘ C` | `Ctrl+Shift+C`（或选中自动复制） |
| 粘贴 | `⌘ V` | `Ctrl+Shift+V`（或右键） |

:::note
在 Linux 终端聚焦时，部分快捷键使用 `Super`（徽标键）而非 `Ctrl`，以避免与 Shell 快捷键冲突。如果 `Ctrl+Shift+F` 无响应，请检查应用内的实际键位绑定。
:::

|------|-------|-----------------|

## SFTP

| 操作 | 快捷键 |
|------|--------|
| 上传 / 下载 | `→` / `←` |
| 全选 | `⌘ A` / `Ctrl+A` |
| 重命名 | `F2` |
| 删除 | `Delete` |

## 小技巧

- **命令面板**（`⌘K` / `Ctrl+K`）提供模糊搜索所有可用操作 — 忘记快捷键时使用它。
- 快捷键绑定与应用内 **设置 → 帮助** 部分和源码中的 `getShortcutCategories` 定义一致。
- 部分终端聚焦快捷键在不同平台有差异，因为 OxideTerm 避免拦截常用 Shell 键位（如 `Ctrl+C` 用于 SIGINT）。
