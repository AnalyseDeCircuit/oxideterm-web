title: 文件管理器
description: 内置本地文件浏览器——支持富媒体预览、属性查看、校验和计算，以及拖拽传输到 SFTP。
---

OxideTerm 内置**文件管理器**，让你无需离开应用即可浏览本地文件系统、使用丰富的文件预览功能，并将文件无缝传输到 SSH 远程端。

## 导航

| 操作 | 功能 |
|------|------|
| 双击文件夹 | 进入目录 |
| 工具栏后退/前进按钮 | 导航历史 |
| 点击路径栏 | 直接输入或粘贴绝对路径 |
| 主页按钮 | 跳转到用户主目录 |

侧边栏收藏夹（⭐）可固定常用目录，一键直达。

## 文件操作

在文件上右键或使用工具栏执行常用操作：

| 操作 | 说明 |
|------|------|
| 新建文件夹 | 在当前目录创建文件夹 |
| 重命名 | 内联重命名（F2 或双击名称） |
| 复制 / 粘贴 | 标准剪贴板操作 |
| 删除 | 移到废纸篓（macOS/Linux）或直接删除 |
| 查看属性 | 文件大小、权限位、创建/修改时间、MD5/SHA256 校验和 |

## 富媒体预览

点击支持的文件类型即可在面板内预览，无需外部应用：

| 类别 | 支持格式 |
|------|---------|
| 图片 | PNG、JPEG、GIF、WebP、SVG、BMP |
| 视频 | MP4、WebM、OGG |
| 音频 | MP3、WAV、OGG、FLAC |
| 代码 | 所有常见纯文本格式（带语法高亮） |
| PDF | 内置 PDF 渲染器 |
| 十六进制 | 任意二进制文件的十六进制转储视图 |
| 字体 | TTF/OTF 字体预览（样本文字） |
| 压缩包 | ZIP/tar 文件内容列表 |
| Office | DOCX/XLSX/PPTX 纯文本提取视图 |

## 属性与校验和

右键 → **属性**显示详细的文件元数据面板：

- 文件大小（字节及人性化单位）
- POSIX 权限（`-rw-r--r--` 符号表示 + 八进制值）
- 所有者 UID/GID
- 创建、修改、访问时间戳
- MD5、SHA-1、SHA-256 校验和（Click to copy）

## 拖拽到 SFTP

打开 SFTP 标签页后，从本地文件管理器**拖拽文件或文件夹**直接放入 SFTP 面板，即可触发上传。反方向同样支持——将远程文件拖回本地文件管理器可触发下载。

传输队列会在 SFTP 标签页底部的传输进度面板中显示进度、速度和预计剩余时间。

## 收藏夹

常用目录可加入收藏夹：

1. 导航到目标目录
2. 点击工具栏中的 ⭐ 图标
3. 文件夹即出现在侧边栏收藏夹列表中

删除收藏：在侧边栏右键收藏项 → **移除收藏**。
