---
title: 广播输入
description: 同时向多个终端发送相同的输入——适用于机群部署和批量运维操作。
---

**广播输入**功能允许你向多个终端**同时发送完全相同的键入内容**，让跨多台服务器的批量操作变得轻快简便。

## 启用广播输入

1. 在工具栏或终端右键菜单中点击**广播**图标（📡）
2. 广播面板弹出，列出所有可用终端（SSH 及本地）
3. **勾选**要加入广播的终端
4. 在**任意**加入广播的终端中键入——所有选中的终端将同步接收输入

广播激活时，每个参与终端的标题栏会显示橙色**广播**指示器。

## 目标选择

| 选项 | 说明 |
|------|------|
| 全选 | 将所有打开的终端纳入广播 |
| 逐一选择 | 在终端列表中手动勾选目标 |
| 按标签筛选 | 仅选中带有指定标签的终端（需先在 Session Manager 中为连接打标签） |

SSH 终端与本地终端**均支持**广播——同一次键入可同时发送到远程服务器和本地 shell。

## 停止广播

再次点击广播图标，或在广播面板中点击**停止广播**，即可退出广播模式。

所有参与终端恢复独立键入模式，各自接收输入互不干扰。

## 使用场景

- **机群同步部署** — 同时向 10 台服务器发送 `apt upgrade -y`
- **配置分发** — 同时编辑多台服务器上的相同配置文件
- **批量重启** — 向所有生产节点广播 `sudo systemctl restart myapp`
- **确认检查** — 在多台主机上同时运行 `df -h` 并对比输出

:::caution
广播输入**不会确认**所有目标终端处于相同的 shell 上下文。在使用前请检查：所有目标终端是否都在期望的目录或 shell 环境中。向不同状态的终端广播可能产生意外结果。
:::
